I built a lightweight web management system for archived medical research project records at FAHZU. Prior to this, I didn't have any experience with API or jQuery - I had only built Flask applications solely based on Python and SQL.

I wished to avoid the potential consequences of the encoding and decoding hassle, so I wrote a function in connect.py to select the correct charset based on specific SQL demands.
